# Reborn — Conférence Live Gratuite : 3 Secrets pour Perdre du Poids

Une landing page premium, moderne et entièrement responsive conçue pour maximiser les conversions d'inscription à un webinaire live gratuit sur la transformation physique.

## Fonctionnalités

- **Design Ultra-Responsive** : Optimisé pour tous les écrans (Smartphones, Tablettes, Ordinateurs de bureau).
- **Esthétique Premium** : Thème sombre moderne combiné à des touches d'or avec des effets de flou d'arrière-plan (*glassmorphism*).
- **Animations Dynamiques** :
  - Effets d'apparition progressive au défilement (*Scroll Reveal*).
  - Micro-animations sur les boutons pour un meilleur retour utilisateur.
- **Carousel de Témoignages** : Curseur interactif supportant le balayage tactile (*swipe*) sur mobile.
- **Navigation Fluide** : En-tête collante (*sticky*) avec détection automatique du scroll pour appliquer des ombres de profondeur, et menu de navigation mobile fluide.

## Technologies Utilisées

- **HTML5** & **CSS3 Vanilla** (Variables CSS, Flexbox, CSS Grid, Requêtes Média)
- **JavaScript (jQuery 3.7.1)** (Scroll reveal personnalisé, gestion tactile du swipe, carousel interactif)
- **FontAwesome 6.5.2** (Pour l'ensemble de l'iconographie moderne)
- **Google Fonts** (Polices *Bebas Neue* et *Inter*)
